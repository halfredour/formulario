# Test Alfredo

## Instalar proyecto

```console
composer install
```

## Borrar cache

```console
php artisan config:cache
```

## Inicializar servidor de pruebas

```console
php artisan serve
```

## Cargar datos de pruebas 

```console
php artisan migrate:refresh --seed
```
